# Jupiter (jupiter-exchange)

Jupiter is the leading Solana DEX aggregator and liquidity infrastructure platform. APIs cover Swap, Tokens, Price, Lend, Trigger (limit orders), Recurring (DCA), Prediction, and Perps. The Developer Platform issues a single API key across products.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/jupiter-exchange/refs/heads/main/apis.yml)
